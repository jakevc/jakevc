Align and count RNA-seq reads with STAR

DNA makes RNA makes Protein, as goes the central dogma of molecular biology. RNA sequencing (RNA-seq) allows researchers to quantify which genes of the DNA are being expressed at any given time, in a given sample of DNA. This tool is very powerful because it allows us to ask questions about the difference in gene expression between different individuals, or between time points in the same individual. RNA-seq produces a text file with millions of sequences we call ‘reads’ that represent the genes they came from in the genome. Read more
